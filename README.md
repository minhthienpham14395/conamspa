# CoNamSpa (Astro)

Scaffold dự án landing page cho CoNamSpa bằng Astro + Tailwind.

Quick start:

```bash
cd conamspa
npm install
npm run dev
```

Build & preview:

```bash
npm run build
npm run preview
```

Hướng dẫn tiếp theo: cài và cấu hình Tailwind nếu cần (`npx astro add tailwind`) hoặc chỉnh `astro.config.mjs`.
